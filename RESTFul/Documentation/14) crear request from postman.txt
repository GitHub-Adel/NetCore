------------------------
1) c# to json:   go to https://csharp2json.io/
2) json to c#:   https://toolslick.com/generation/code/json-to-csharp-class


nota:
---------------------
{} : object: para recibirlo en c#, crea una class and create a object que reciba el json.
[] : collection: para recibirla en c#, create a class and create a collection de esta que reciba el json.
     ojo las propiedades de las class en c# deben ser iguales que la del json.
