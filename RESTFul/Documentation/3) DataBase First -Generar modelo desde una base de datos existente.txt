https://www.nuget.org/packages
----------------------------
1) tener instalado en SocialMedia.Api:
	Microsoft.EntityFrameworkCore.SqlServer
	Microsoft.EntityFrameworkCore.Design
	dotnet-ef

   tener instalado en SocialMedia.Infrastructure:
        Microsoft.EntityFrameworkCore.SqlServer
	Microsoft.EntityFrameworkCore

2) Asegurarse que el project compile sin error con: dotnet build

3) run command: dotnet ef dbcontext scaffold "CONNECTION_STRING" Microsoft.EntityFrameworkCore.SqlServer -context-dir CARPETA_DESTINO_CONTEXTO -o CARPETA_DESTINO_MODELOS

4) to delete OnConfiguring method from dbcontext.

5) to config. connectionString into appsettings.json

6) to say al context que use la connectionString que esta en el appsetting.json, a traves del Startup.cs
   services.AddDbContext<SocialmediaDBContext>(x => x.UseSqlServer(Configuration.GetConnectionString("SocialMedia")));

fin
