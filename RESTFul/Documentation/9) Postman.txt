5) show a request type (Get,Post) etc.
6) copy url https://localhost:5001
4) add Route to url https://localhost:5001/api/ControllerName


convert c# to json
------------------------
1) go to https://csharp2json.io/
2) copy c# class
3) press a json button.
4) press a copy button.
